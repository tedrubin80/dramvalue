System Design

Six-Container Architecture

Docker Compose orchestrates the full stack: an async FastAPI application, Celery workers for distributed scraping, a beat scheduler for cron-like task management, PostgreSQL for persistence, Redis as a message broker, and a Tor proxy for anonymized scraping.

⚙

FastAPI Application

Async request handling with Jinja2 SSR templates. Serves the web frontend and REST API on a single uvicorn process with 4 workers.

FastAPI SQLAlchemy 2.0 Jinja2 Pydantic v2
🕷

Celery Workers

Distributed task queue running 14 Scrapy spiders on staggered schedules. Handles scraping, normalization, and maintenance jobs across 3 queues.

Celery Scrapy Playwright Beat Scheduler
🗂

PostgreSQL 16

Async connection pooling via asyncpg. 12 ORM models with composite indexes, unique constraints, and full-text search support.

asyncpg Alembic 12 Models

Security Layer

JWT auth with httpOnly cookies, rate limiting via slowapi, 6 security headers, CORS lockdown, and admin-gated health endpoints.

JWT + bcrypt slowapi HSTS CSP

Redis Broker

Message broker for Celery task distribution. Handles task serialization, result backend, and rate limiting state.

# Shopify-based retail scraper with anti-bot evasion

class DekantaSpider(Spider):
    name = "dekanta"
    custom_settings = {"CONCURRENT_REQUESTS": 2}

    def start_requests(self):
        # Paginate through Shopify's products.json API
        for page in range(1, 50):
            yield Request(
                url=f"https://dekanta.com/products.json?page={page}",
                callback=self.parse_products,
                meta={"page": page},
            )

    def parse_products(self, response):
        data = response.json()
        for product in data["products"]:
            yield RetailPriceItem(
                source_name="Dekanta",
                bottle_name=product["title"],
                price=product["variants"][0]["price"],
                currency="JPY",
                source_url=f"https://dekanta.com/products/{product['handle']}",
            )
# 4-stage Scrapy pipeline with fuzzy bottle matching

class NormalizationPipeline:
    """Matches scraped items to existing bottles using RapidFuzz."""

    def process_item(self, item, spider):
        normalized = normalize_bottle_name(item["bottle_name"])

        # Try exact match first, then fuzzy
        match = self.exact_matches.get(normalized)
        if not match:
            result = process.extractOne(
                normalized,
                self.bottle_names,
                scorer=fuzz.token_sort_ratio,
                score_cutoff=85,
            )
            if result:
                match = self.name_to_bottle[result[0]]

        if match:
            item["bottle_id"] = match.id
        else:
            # Auto-create new bottle record
            bottle = await create_bottle(normalized, item)
            item["bottle_id"] = bottle.id

        return item
# Async FastAPI route with SQLAlchemy + Jinja2 SSR

@router.get("/market", response_class=HTMLResponse)
async def market_page(
    request: Request,
    db: AsyncSession = Depends(get_db),
):
    # Aggregate monthly trading volume across all houses
    monthly = await db.execute(
        select(
            MarketStat.period_date,
            func.sum(MarketStat.trading_volume).label("volume"),
            func.avg(MarketStat.winning_bid_mean).label("avg_bid"),
            func.sum(MarketStat.lots_count).label("lots"),
        )
        .group_by(MarketStat.period_date)
        .order_by(MarketStat.period_date)
    )

    # Auction house leaderboard by total volume
    leaderboard = await db.execute(
        select(
            MarketStat.auction_name,
            func.sum(MarketStat.trading_volume).label("total"),
        )
        .group_by(MarketStat.auction_name)
        .order_by(func.sum(MarketStat.trading_volume).desc())
    )

    return templates.TemplateResponse("market.html", context)
# Security middleware: headers, rate limiting, CORS lockdown

class SecurityHeadersMiddleware:
    async def __call__(self, scope, receive, send):
        async def send_with_headers(message):
            if message["type"] == "http.response.start":
                message["headers"] += [
                    (b"x-content-type-options", b"nosniff"),
                    (b"x-frame-options", b"DENY"),
                    (b"strict-transport-security",
                     b"max-age=31536000; includeSubDomains"),
                    (b"referrer-policy",
                     b"strict-origin-when-cross-origin"),
                ]
            await send(message)
        await self.app(scope, receive, send_with_headers)

Platform Features

What It Does

🔍 Full-Text Search

Search 9,600+ bottles by name, distillery, or category with ILIKE queries and input sanitization against LIKE injection.

📈 Price History Charts

Chart.js visualizations showing auction trends, price movements, and monthly trading volumes across all houses.

📊 Market Analytics

Aggregate market stats from 27 auction houses: $892M total volume, 3M+ lots tracked from 2005-2023.

📦 Data Export

CSV and JSON export endpoints with rate limiting. Filter by category, date range, and price bounds.

👤 User Accounts

JWT authentication with bcrypt hashing, httpOnly cookies, user collections, and price alert subscriptions.

🤖 AI Research

Perplexity-powered endpoints for bottle research, market analysis, value estimation, and side-by-side comparison.

📖 Brand Catalog

Browse 4,800+ whisky brands by category (Scotch, Bourbon, Irish, Japanese) with pagination and search.

🔥 Trending Dashboard

Real-time trending bottles based on recent auction activity, price velocity, and submission frequency.

🛠 Admin Panel

Scraper management, moderation queue, manual spider triggering, and data quality monitoring.
